# 🚀 Zentryx Labs

### AI-Powered Liquidity Intelligence Platform

Zentryx Labs is a modern SaaS-grade platform designed to decode global liquidity flow and market behavior using AI-driven insights and real-time analytics.

👉 https://codebyalpesh.github.io/zentryx-labs/

## 🧠 Overview

Zentryx Labs helps users understand **where money is moving in the market** — across assets, sectors, and global systems.

Built with a focus on:

* Clean SaaS UI/UX
* High-performance frontend
* Scalable design system
* Real-time visual intelligence

---

## ✨ Features

* 📊 AI-powered liquidity insights
* ⚡ Smooth scroll & reveal animations
* 📈 Interactive dashboard UI
* 🎯 Conversion-focused SaaS layout
* 📱 Fully responsive design
* 🎨 Design system based architecture (tokens, utilities)

---

## 🏗️ Tech Stack

* HTML5
* CSS3 (Design System + Tokens)
* JavaScript (Vanilla)
* Scroll-based animations
* Responsive layout system

## 🎨 Design Philosophy

Zentryx Labs follows a **SaaS-first design DNA**:

* Minimal but powerful visuals
* High contrast + readability
* Motion-driven interactions
* Component-based layout system
